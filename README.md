# 🚗 CarBook — Car Rental Platform

A full-stack MERN car rental web application where users can browse, 
filter and book cars with secure online payments.

## 🛠️ Tech Stack

**Frontend:** React.js, Tailwind CSS, React Router, Axios  
**Backend:** Node.js, Express.js, MongoDB, Mongoose  
**Auth:** JWT, bcrypt, Google OAuth 2.0 (Passport.js)  
**Payments:** Razorpay  
**Storage:** Cloudinary (image uploads)  
**Email:** Nodemailer (Gmail SMTP)  

## ✨ Features

- 🔐 Email/Password & Google OAuth authentication
- 🚗 Browse cars with filters (category, price, fuel, transmission)
- 📅 Book cars hourly, daily, weekly or via fixed packages
- 💳 Secure payments via Razorpay with signature verification
- 📧 Email notifications for booking confirmation & cancellation
- ⭐ Reviews and ratings system
- 👨‍💼 Admin dashboard — manage cars, bookings and view stats
- 📱 Fully responsive — mobile, tablet and desktop
